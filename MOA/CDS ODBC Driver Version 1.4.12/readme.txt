THIS DISTRIBUTION CONTAINS THE FOLLOWING SOFTWARE:


   CDS
   -----------
      MOA CDS for NT, Version 1.4.10 (85-35011-D02)

   ODBC Driver
   -----------
      MOA CDS ODBC Driver, Version 1.4.10 (85-35023-D02)


   WordView
   -----------
      Microsoft Word read-only document viewer, Version 8.0.0.0
      This package is needed only to read documents on Windows NT 3.51.
      It is generally not needed on Windows 95 or Windows NT 4.0.
      It is never needed except to read the documentation that is 
      intended to inform developers developing with mosaic OA and CDS.


MOA CDS 1.4.10 Installation
==========================

1) Run SRVSETUP.BAT to install cds server package V1.4.10.
   Reboot when prompted to do so to allow application environment variables
   to be set.

** For additional information, please refer to the CDS Installation 
** Notes (readme.DOC) contained in directory \moacdsv\disk1.



MOA CDS ODBC Driver 1.4.10 Installation
======================================

1) Run SETUP.BAT to install cds odbc driver package V1.4.10.
   Reboot when prompted to do so to allow application environment variables
   to be set.

** For additional information, please refer to the CDS ODBC Driver Installation
** Notes (readme.DOC) contained in directory \cdsodbc\disk1.


WordView 8.0 Installation
======================================
1) cd wordview\disk1
2) run setup.exe.
